# WP REST API V2 Menus
Adding menus endpoints on WP REST API v2

## Routes
Get all menus:

    GET /menus/v1/menus
    
Get menus data from slug:

    GET /menus/v1/menus/{slug}

Get all locations:

    GET /menus/v1/locations
    
Get menus data from location's slug

    GET /menus/v1/locations/{slug}
